The Atlassian Bigram Index. An interactive bigram-to-glyph lookup tool for No Man's Sky, built on the Glyph Rosetta Dataset.

Every learnable word across all five alien languages in No Man's Sky maps to one of the game's 16 portal glyphs via its first two letters. This orthographic mapping holds across 4,820 documented words and 185 unique bigrams without exception. Because of the large amount of available bigrams, most English words will also map to a glyph, not just the words learnable in game.

How To Use: 

Enter any word or phrase to see its glyph assignments and hex portal output in real time. The tool maps glyphs based on the first two letters of each word. A reverse lookup tab accepts any hex portal address and displays the corresponding glyphs. Hovering over each glyph reveals its assigned bigrams, which can be used to build your own phrase that produces the same address. The tool works fully offline as a standalone HTML file.

Known Issues:

A cosmetic edge case exists in the hex output colour coding when a missing bigram occurs at or past the 12 word threshold. Glyph assignments and hex values are not affected. Only the colours display incorrectly in this specific scenario.
